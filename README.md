# Rust Learn By Practicing

Cada crate es un mini proyecto pensado para ayudarme a aprender Rust.
Each crate is a mini project designed to help me learn Rust.

## Crates ordenados de menor a mayor complejidad / Crates ordered from least to most complex

1. **hello-world** - Primer programa que imprime "Hello, world!". / First program that prints "Hello, world!".
2. **variables** - Introducción a las variables en Rust, declaración e impresión. / Introduction to variables in Rust: declaration and printing.
3. **tipos-primitivos** - Ejemplos de tipos de datos primitivos (`i32`, `bool`, etc.). / Examples of primitive data types (`i32`, `bool`, etc.).
4. **variables-mutabilidad** - Demostración de la inmutabilidad por defecto y cómo hacer variables mutables. / Demonstrates default immutability and how to make variables mutable.
5. **control-flujo** - Uso de estructuras de control como `if`, `loop`, `while`, y `for`. / Usage of control structures like `if`, `loop`, `while`, and `for`.
6. **funciones** - Definición y llamada de funciones, paso de parámetros, y retorno de valores. / Defining and calling functions, passing parameters, and returning values.
7. **structs** - Definición y uso de estructuras (`struct`) para agrupar datos. / Defining and using structs to group data.
8. **slices** - Trabajo con slices y referencias a partes de arrays o vectores. / Working with slices and references to parts of arrays or vectors.
9. **ownership** - Concepto de propiedad, movimiento, y `drop`. / Ownership concept, moves, and `drop`.
10. **borrowing** - Prestamos inmutables y mutables, reglas de préstamo. / Immutable and mutable borrowing, borrowing rules.
11. **enums** - Definición y uso de enumeraciones para representar valores discretos. / Defining and using enums to represent discrete values.
12. **options** - Manejo de valores opcionales con `Option<T>`. / Handling optional values with `Option<T>`.
13. **Result** - Manejo de errores con `Result<T, E>`. / Error handling with `Result<T, E>`.
14. **vecs** - Trabajo con vectores dinámicos (`Vec<T>`). / Working with dynamic vectors (`Vec<T>`).
15. **hashmaps** - Uso de mapas hash para almacenar pares clave-valor. / Using hash maps to store key-value pairs.
16. **iterators** - Iteración sobre colecciones usando iteradores. / Iterating over collections using iterators.
17. **Traits** - Definición y implementación de traits para compartir comportamiento. / Defining and implementing traits to share behavior.
18. **dyn-traits** - Uso de traits dinámicos con `dyn Trait`. / Using dynamic traits with `dyn Trait`.
19. **generics** - Programación genérica con parámetros de tipo. / Generic programming with type parameters.
20. **basic-patter-matching** - Coincidencia de patrones básica con `match`. / Basic pattern matching with `match`.
21. **pattern-matching-avanzado** - Técnicas avanzadas de coincidencia de patrones. / Advanced pattern matching techniques.
22. **implementation-funciones-asociadas** - Implementación de funciones asociadas y métodos. / Implementing associated functions and methods.
23. **modules** - Organización del código con módulos. / Organizing code with modules.
24. **libexample** - Creación de una biblioteca de ejemplo. / Creating an example library.
25. **libexample-usage** - Uso de bibliotecas externas. / Using external libraries.
26. **operador-propagacion** - Uso del operador `?` para propagación de errores. / Using the `?` operator for error propagation.
27. **text** - Manejo de cadenas de texto y caracteres. / Handling text strings and characters.
28. **closures** - Definición y uso de closures (funciones anónimas). / Defining and using closures (anonymous functions).
